# Learning Tasks

## Network Programming: Learning Tasks & Datasets

| Learning Task | Task Description | Relevant Dataset / Link |
| --- | --- | --- |
| **Network Intrusion Detection & DDoS Mitigation** | Detect whether network traffic is normal or malicious using packet- and flow-level features. | [Kaggle: eBPF XDP Tracing](https://www.kaggle.com/datasets/ngoquy/ebpf-network-tracing-with-xdp-layer-3-and-4-only) |
| **Volumetric Flood Identification (DDoS)** | Identify large-scale flooding attacks such as SYN or UDP floods. | [BoT-IoT Dataset](https://research.unsw.edu.au/projects/bot-iot-dataset) |
| **Traffic / Application Classification** | Classify applications based on traffic patterns, even when payloads are encrypted. | [UNIBS Traffic Traces](http://netweb.ing.unibs.it/~ntw/tools/traces/) |
| **Adaptive Bitrate (ABR) Prediction** | Predict available bandwidth to choose an appropriate video quality. | [Stanford Puffer Results](https://puffer.stanford.edu/results/) |
| **QoS Class Identification** | Group traffic into priority classes such as voice, video, or background traffic. | [ISCX VPN-nonVPN (2016)](https://www.unb.ca/cic/datasets/vpn.html) |
| **Network Event Classification** | Identify the cause of network failures from logs and traffic patterns. | [IEEE Failure Type Research](https://ieeexplore.ieee.org/document/9474482) |
| **Routing Decision Classification** | Learn which network path performs better under given conditions. | [Kaggle: Multi-Criteria Routing](https://www.kaggle.com/datasets/ziya07/multi-criteria-network-routing-dataset) |
| **Congestion State Classification** | Classify the current congestion level of the network. | *Task: Multi-class Classification* |
| **Queue Length Prediction** | Predict buffer occupancy to reduce packet loss. | *Task: Regression* |
